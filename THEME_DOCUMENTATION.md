# Theme Documentation

## Button Styles

### Primary Buttons
- **Class**: `.btn-primary` or inline style with `var(--accent-gold)` background.
- **Appearance**: Solid gold background, dark text (`var(--text-on-gold)`), no border.
- **Usage**: Main call-to-action buttons (e.g., "Need Instant Help?", "Explore Crime Types").
- **Update**: Both home page hero buttons are now standardized to this style for consistency.

### Secondary Buttons
- **Class**: `.btn-outline-primary`
- **Appearance**: Transparent background, gold border, gold text.
- **Usage**: Alternative actions or secondary links.

## Color Palette
- **Gold**: `#c5a059` (Accent)
- **Dark Blue**: `#020617` (Background)
- **Text**: `#ffffff` (Primary), `#94a3b8` (Secondary)
